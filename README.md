# dotfiles
Dotfiles and configs I use for my Arch Linux setup.
## Stuff I use
- Window manager: dwm (uselessgaps, alpha)
- Compositor: Compton
- Editor: Vim
- Terminal: st (alpha)
- Theming: Pywal
- Shell : Fish
